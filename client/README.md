# WebSocket Chat - Client

Це клієнтська частина чат-додатку на Angular.

## Як запустити проект

Найпростіший спосіб запустити весь проект (клієнт + сервер + БД) — використовувати Docker Compose з кореневої директорії проекту.

### Запуск через Docker (рекомендовано)

1. Переконайтеся, що у вас встановлено **Docker** та **Docker Compose**.
2. Відкрийте термінал у кореневій папці проекту (де знаходиться файл `docker-compose.yml`).
3. Виконайте команду:
   ```powershell
   docker-compose up --build -d
   ```
4. Після завершення білду клієнт буде доступний за адресою: [http://localhost:4200](http://localhost:4200)

---

## Доступ у локальній мережі

Проект налаштовано для роботи у вашій домашній мережі.

### Доступ з інших пристроїв
Щоб зайти в чат з іншого пристрою (телефону або другого ноутбука), підключеного до того ж Wi-Fi, використовуйте адресу:
**[http://192.168.0.198:4200](http://192.168.0.198:4200)**

### Важливо для розробки
Якщо ваша локальна IP-адреса зміниться, вам потрібно оновити її у файлі:
`src/app/core/services/socket.service.ts`

```typescript
this.socket = io('http://ВЛАСТИВА_IP_АДРЕСА:3000');
```

## Розробка (локальний запуск без Docker)

Якщо ви хочете запустити клієнт окремо:

1. Перейдіть у папку `client`:
   ```powershell
   cd client
   ```
2. Встановіть залежності:
   ```powershell
   npm install
   ```
3. Запустіть сервер розробки:
   ```powershell
   npm start
   ```
4. Клієнт відкриється на `http://localhost:4200`.

> [!NOTE]
> При локальному запуску без Docker переконайтеся, що бекенд-сервер запущений на порту 3000.
